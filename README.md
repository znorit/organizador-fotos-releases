# 📷 Organizador de Fotos y Vídeos para Windows

Aplicación de escritorio para **organizar, clasificar y limpiar** tu colección de fotos y vídeos en Windows.

---

## ✨ Características principales

- 📁 **Organización automática** por fecha, cámara o carpeta personalizada
- 🔍 **Detección de duplicados** exactos y similares (pHash + BK-Tree)
- 🖼️ **Visor integrado** con comparativa lado a lado
- 🎞️ **Soporte de vídeo** (MP4, MKV, AVI, MOV y más)
- 📱 **Soporte HEIC/HEIF** (fotos iPhone)
- 📊 **Informes PDF** del proceso de organización
- 🔄 **Actualizaciones automáticas** desde esta página

---

## 💻 Requisitos del sistema

- Windows 10 / 11 (64 bits)
- 4 GB RAM mínimo recomendado
- Sin necesidad de instalar Python ni dependencias

---

## ⬇️ Descarga e instalación

1. Ve a la sección [**Releases**](../../releases/latest)
2. Descarga el archivo `OrganizadorFotos_vX.X.X_Setup.exe`
3. Ejecuta el instalador y sigue los pasos
